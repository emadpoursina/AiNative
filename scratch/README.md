# Scratch

Gitignored working area for agents and humans. Put temp files, drafts, pasted logs, handoff notes, and anything you want in the repo workspace but not in git.

**Rules**

- Nothing here is canonical — promote worth keeping into `docs/` (or the right layer), then delete
- Agents may read and write here freely
- Do not commit contents of this folder (only this README is tracked)

See [ENGINEERING-OS.md](../ENGINEERING-OS.md#scratch-scratch) for the Friday review ritual.
