# Team-specific documentation

Add one folder per team you onboard, for example:

```text
teams/acme-product/README.md
```

Include:

- Stack and repos
- Branch names (if different from the default release system)
- Task board tool and link
- Meeting cadence and attendees
- Who approves production releases
- Any adapted checklists

Universal workflows stay in [`../systems/`](../systems/).
